Smart Library - An automated library with student interaction

Developed an RFID-based automated library management system that streamlines book issuance, returns, and theft detection. The system uses RFID tags on books and user cards for secure, real-time tracking of transactions, integrated with a centralized database and a web portal. Key features include automated attendance, contactless book return via deposit box, theft alerts at the exit gate, and user-friendly interfaces for students and staff. Technologies used: Arduino, RFID (MFRC522), ESP8266 Wi-Fi Module, MySQL, PHP, HTML, CSS, and JavaScript.
